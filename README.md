# Apartments
Часть функционала сайта недвижимости в виде фильтрации списка квартир в новостройке
## Функционал:
* Адаптивная верстка HTML, CSS, JS по методологии БЭМ
* API
* Загрузка квартир из файла .json
* Просмотр квартир по фильтрам
* Сортировка данных
